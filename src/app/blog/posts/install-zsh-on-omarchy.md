title: 'Change shell to Zsh on Omarchy'
date: '2026-05-05'
status: 'in-progress'
tag: 'linux'
---

You change the shell on Omarchy by running `chsh -s $(which zsh)` after installing zsh. After that you need to run `sudo reboot` for zsh to be activated.


Appreciate this [discussion](https://askubuntu.com/questions/131823/how-to-make-zsh-the-default-shell) for the tip.
